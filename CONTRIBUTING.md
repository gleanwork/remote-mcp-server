# Contributing to Glean Remote MCP Server

Thank you for your interest in contributing to the Glean Remote MCP Server project!

## About This Repository

This repository is a **metadata-only repository** that serves as the public home for Glean's MCP server registration in the [MCP Registry](https://registry.modelcontextprotocol.io). It contains:

- `server.json` - Registry metadata for the Glean MCP Server
- GitHub Actions workflow for publishing to the MCP Registry
- Documentation about the publishing process

The actual MCP server functionality is hosted by Glean and is not open source.

## How to Contribute

### Reporting Issues

If you encounter issues with the Glean MCP Server:

- **Server functionality issues**: Please contact Glean support through your organization's support channel
- **Documentation issues**: Open an issue in this repository or submit a pull request
- **Registry metadata issues**: Open an issue in this repository

### Suggesting Features

Feature requests for the Glean MCP Server should be directed to:

- Your organization's Glean support channel
- [Glean's documentation feedback](https://docs.glean.com)

### Pull Requests

We welcome pull requests for:

- Fixing typos or improving documentation
- Updating links or references
- Improving the README or other documentation files

For any changes to `server.json`, please open an issue first to discuss the proposed changes.

## Pull Request Process

1. Fork the repository
2. Create a feature branch (`git checkout -b fix/typo-in-readme`)
3. Make your changes
4. Commit with a clear message describing the change
5. Push to your fork and open a pull request

## Questions

If you have questions about contributing, please open an issue in this repository.
